## PART 8：リスク管理ルール

### 日立株のリスク管理

```
⚠️ 株価が4,500円以下に下落した場合：
   → 指値を現在値近辺に修正して早期売却を検討
   → 含み益が減る前に転換を加速

⚠️ 好材料・急騰があった場合：
   → 積極的に売却（高値を活かす）
   → 年1,000万円の上限を一時的に引き上げ検討

⚠️ 悪決算・不祥事が発生した場合：
   → 即時に指値を撤回して成行き売却を検討
```

### 為替リスク管理

```
円高（130円以下）になった場合：
→ VOOの新規ドル買い増しは一時停止
→ 円建てTOPIX ETFの積立を増額

円安（160円以上）になった場合：
→ ドル建て資産の新規購入は抑制
→ 既存のVOO・LQDはそのまま保有継続
```

### 金利前提・逆シナリオ

```
【ベースケース】
・日本：利上げ方向を基本線に置く
・米国：利下げ予想が強いが、上振れリスクもある
・日米ともインフレ圧力は残り、対策が必要という前提で考える

【逆シナリオ】
・日本：景気悪化やインフレ鈍化で利上げが止まる可能性
・米国：インフレ再燃や景気底堅さで高金利長期化の可能性
・日米同時にインフレがしぶとく残るケースも想定する

【運用上の含意】
・個人向け国債（変動10年）は円建ての安全資産・待機資金として有効
・LQD/BNDは金利上昇局面で価格変動が出るため、買い方は分割を基本にする
・金利見通しを1本に決め打ちせず、現金・短期債・変動金利で耐性を持たせる
```

### 1-3) 過去の暴落で起きたこと（代表例）

```
【2008年 金融危機】
・米国株はピークから -50%級
・REITは -60%級の大幅下落
・投資適格社債（LQD相当）も一時大きく下落
・一方で米国債は上昇し、債券と現金の防御力が有効

【2020年 コロナショック】
・約1か月で世界株が -30%前後
・社債/REITも急落したが、金融緩和で回復が早かった
・現金を持っていた投資家は段階買いが可能だった

【2022年 インフレ・急速利上げ】
・株式と債券が同時安（分散が効きにくい年）
・長期債は逆風、REITも軟調
・この局面では「現金比率」と「段階的リバランス」が有効
```

### 1-4) リバランスの効果（なぜ必要か）

```
効果①：リスクの暴走を防ぐ
  上がり続けた資産の比率を下げ、集中リスクを抑制する。

効果②：機械的に「高く売って安く買う」
  目標比率に戻す過程で、過熱資産を利確し、下落資産を買い増す。

効果③：生活防衛と取り崩し安定化
  現金・債券を維持することで、暴落時に株の狼狽売りを減らす。
```

#### リバランスの簡易イメージ

```
前提：現金10%を別枠、投資資産90%を配分
ケース：株式急落で VOO/TOPIX が目標比率を下回る

対応：
1) 新規配当と現金クッションの一部で不足分を補充
2) 債券（LQD/BND）が相対的に上振れなら一部売却して株を補充
3) 1回で買わず3分割で執行（タイミング分散）

結果：
・平均取得単価を引き下げやすい
・回復局面でポートフォリオ全体の戻りが早くなる
・最大下落率（ドローダウン）を抑えやすい
```

```
① 目標比率から ±5% 以上乖離
② 相対乖離が 25% 以上（例：目標8%が10%以上 or 6%以下）
③ 単一銘柄・単一ETFがポートフォリオの 55% 超
④ 現金比率が 6% 未満 or 15% 超
```

### 3) 執行順序（税金最小化ルール）

```
Step 1：新規入金・配当金でアンダーウェイト資産を補充
Step 2：NISA枠は最も不足している「コア資産（VOO/TOPIX）」を優先
Step 3：特定口座の売却は「含み益率が低いロット」から実施
Step 4：年間の課税売却益は税務予算内に抑制（目安：年300万円まで）
```

### 4) 配当金と現金クッションの運用ルール

```
配当金の使途：
  80% 生活・寄付・自由支出
  20% 現金クッション補充

現金比率の運用：
  平常時：8〜10%
  警戒時：12〜15%（円高進行・高ボラ・減配懸念）
  下限：6%（割り込んだら新規投資を一時停止して現金回復）
```

### 4-1) 生活防衛資金

```
・生活防衛資金は1年分を目安に、500万〜1,000万円を確保する
・用途は急な支出・収入減・相場急変への備えに限定する
・置き場所は現金を基本とし、一部を個人向け国債（変動10年）で補完する
・投資資金と混同せず、売却不要で1年以上保てる資金として扱う
```

### 5) 相場急変時の例外ルール

```
株式市場が高値から20%以上下落した場合：
  ・売却リバランスは最小化
  ・現金クッションの範囲内で、3分割（当月・翌月・翌々月）で買い増し
  ・買い増し対象は「最も目標比率を下回るコア資産」を優先

円高（130円以下）時：
  ・ドル建て新規買付を抑制
  ・TOPIX ETFの補充比率を引き上げ

円安（160円以上）時：
  ・ドル建ての新規一括投資を避け、毎月分割で執行
```

### 6) 年次メンテナンス（毎年12月）

```
□ 実績比率と目標比率の乖離を計測
□ 翌年の売却税見込み（概算）を更新
□ 配当実績（税引後）と生活費・寄付比率を見直し
□ 使途不明の現金を再配分（現金上限15%を超える場合）
□ 10年期待リターンと下方シナリオを再計算
```

### 7) ガードレール（禁止事項）

```
× テーマだけで売買しない（必ず比率とルールで判断）
× 1資産へ15%以上の新規追加を単月で実行しない
× 現金比率6%未満で高リスク資産を買い増ししない
× 損失回避のための無期限塩漬けをしない（年2回で機械判定）
```
