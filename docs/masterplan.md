```
【鉄の掟】
① 日立株は毎年1,000万円を上限に売却

③ 感情で動かない。ルール通りに機械的に実行
④ 毎年1月を売却実行月とする（年次ルール化）
⑤ 決算発表（7〜8月・10〜11月）前後に指値を調整

【税金ルール】
・年間売却益 約800万円 × 20.315% ≈ 約163万円/年
```

## PART 8：リスク管理ルール

### 日立株のリスク管理

```
⚠️ 株価が4,500円以下に下落した場合：
   → 指値を現在値近辺に修正して早期売却を検討
   → 含み益が減る前に転換を加速

⚠️ 好材料・急騰があった場合：
   → 積極的に売却（高値を活かす）
   → 年1,000万円の上限を一時的に引き上げ検討

⚠️ 悪決算・不祥事が発生した場合：
   → 即時に指値を撤回して成行き売却を検討
```

### 為替リスク管理

```
円高（130円以下）になった場合：
→ VOOの新規ドル買い増しは一時停止
→ 円建てTOPIX ETFの積立を増額

円安（160円以上）になった場合：
→ ドル建て資産の新規購入は抑制
→ 既存のVOO・LQDはそのまま保有継続
```

### 楽天銀行証券担保ローン（緊急時のみ）

```
サービス開始：2025年6月22日
金利：1.875%（1,000万円超）
融資上限：担保評価額の60%
⚠️ 投資目的には使用不可
⚠️ NISA口座の株は担保対象外
✅ 緊急の生活資金需要時の「株を売らない」手段として活用
日立株（特定口座）担保で最大約3,257万円まで借入可能
```
