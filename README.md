# 📊 ポートフォリオ マスタープラン

> 日立株売却による最適資産転換 5ヵ年計画

## 概要
個人株式ポートフォリオの最適化計画を管理するリポジトリです。
含み益の大きい日立株を、家族（妻・子3人）への暦年贈与（年110万円の基礎控除）を組み合わせながら段階的に売却し、世帯全体の税負担を最適化しつつグローバル資産へ転換する5ヵ年計画です。
GitHub Copilot / Claude の AI アシスタントが6名の投資専門家（税理士・数理統計ペルソナ含む）として討論形式でアドバイスを提供します。
