# 📊 ポートフォリオ マスタープラン
日立株売却による最適資産転換 5ヵ年計画

## 概要
個人株式ポートフォリオの最適化計画を管理するリポジトリです。
含み益の大きい日立株を、家族（妻・子3人）への暦年贈与（年110万円の基礎控除）を組み合わせながら段階的に売却し、世帯全体の税負担を最適化しつつ、グローバル資産（ドル比率70%目標）へ転換する5ヵ年計画です。

1ドル150円台という歴史的な円安局面における「為替リスクの制御」と、10年以上に及ぶ家族口座への「安全な資産承継」を両立させるため、GitHub Copilot / Claude の AI アシスタントが**8名の投資専門家（ファミリーオフィス・プライベートバンキング委員会）**として討論形式でアドバイスを提供します。
